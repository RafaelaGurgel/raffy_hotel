# Raffy Hotel - Sistema de Reservas

Sistema de reservas para hotel boutique, feito com Python e Flet.

## Funcionalidades

- Gerenciamento de clientes (adicionar, visualizar)
- Cadastro e listagem de quartos
- Criação e cancelamento de reservas
- Interface interativa e intuitiva no navegador

## Como rodar

1. Instale Python (>=3.10)
2. Clone ou baixe este repositório
3. Instale dependências:

```bash
pip install -r requirements.txt
