#  Raffy Hotel - Sistema de Reservas

Sistema de reservas para hotel boutique, desenvolvido em **Python** usando **Flet**, com interface moderna e interativa no navegador.

---

##  Funcionalidades

- Gerenciamento de clientes: adicionar e visualizar clientes cadastrados.
- Cadastro e listagem de quartos com imagens, preços e disponibilidade.
- Criação e cancelamento de reservas com datas de check-in e check-out.
- Interface interativa e responsiva, com feedback visual e cores harmoniosas.

---

##  Visual

- Cards de quartos com imagem, nome, tipo e status (Disponível/Ocupado).
- Botões grandes e chamativos para navegar entre telas.
- Layout responsivo, funcionando em desktop e mobile.

---

##  Tecnologias

- Python 3.10 ou superior  
- [Flet](https://flet.dev/) para criar interface web moderna
